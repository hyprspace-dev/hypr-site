PHASE 2 — INTERACTIVE ELEMENTS
────────────────────────────────

4. ThemePicker.jsx (Preact island)
   - Renders in the navbar as 4 color dots: ● ● ● ●
   - Presets: Cyan (#00d4ff), Magenta (#f472b6), Green (#34d399), Gold (#fbbf24)
   - Swaps CSS variable --accent on :root via document.documentElement.style
   - Persists choice in localStorage so it survives page reloads
   - Tailwind config must use CSS variables so swap propagates everywhere

5. GitHubStats.jsx (Preact island)
   - Fetches https://api.github.com/repos/YOUR_USERNAME/hyprspaceos
   - Displays: ★ stars, ◔ forks, last commit date
   - Renders as a subtle badge bar in the footer or hero
   - Cache response in sessionStorage to avoid rate limiting

6. ConfigViewer.jsx (Preact island)
   - A horizontally-scrollable, syntax-highlighted window
   - Shows 20-30 lines from a real hyprland.conf with line numbers
   - Animated line-by-line reveal when scrolled into view
   - Monospace font, dark background, green prompts
   - Include actual configs: blur, animations, window rules, keybinds

    /* Reduced motion */

      *, *::before, *::after {
        animation-duration: 0.01ms !important;
        transition-duration: 0.01ms !important;
      }
    }

    /* Scanline overlay utility */
    .scanlines::after {
      content: '';
      position: absolute; inset: 0;
      background: repeating-linear-gradient(
        0deg, transparent, transparent 2px,

      );
      pointer-events: none;
    }

    - Respect prefers-reduced-motion

DESIGN NOTES
─────────────

  - No centered symmetric card grids. Everything should feel slightly offset.
  - Full-bleed sections between content blocks for visual breathing room.
  - Terminal aesthetic (monospace, green-on-black, blinking cursor) is a
    recurring visual motif that ties the distro identity to the site.
  - Real numbers beat marketing language every time.
  - Interactive elements (theme picker, stats) build trust and show the
    project is alive and maintained.
  - Every placeholder/generic element is an opportunity for authenticity.
